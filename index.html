Where your data lives

Every change is saved instantly to this browser's local storage, so nothing is lost on refresh — even if you never connect a Google Sheet. Connecting a sheet below adds a second copy that is kept in sync automatically, so you can view/back up the data in Google Sheets and access it from other devices.

Google Sheet sync

This app can mirror all client and food-log data to a Google Sheet through a small Google Apps Script "web app" that acts as the connector. Paste your deployed web app URL below, then use Load to pull existing data and Save to write current data back to the sheet. Once a URL is saved, every add/edit/delete auto-syncs a few seconds later. Setup instructions are provided separately — see the message alongside this app.

Auto-fill food nutrition (USDA database)

Add a free USDA FoodData Central API key to look up calories/protein/carb/fat automatically while typing a food name in the Food Log tab — no more manual entry for common foods. Get a free key at fdc.nal.usda.gov/api-key-signup.html (instant, just an email). Coverage is strongest for US / packaged / generic foods; dishes without a close match can still be entered manually.

About auto-calculated targets

Targets use the Mifflin-St Jeor formula for resting metabolic rate, scaled by activity level for total daily energy expenditure, then adjusted by goal: lose fat (-500 kcal, 2.2 g/kg protein), gain muscle (+350 kcal, 2.0 g/kg protein), maintain (2.0 g/kg protein), or recomposition (-200 kcal, 2.2 g/kg protein). Fat is set to 25–30% of calories and the remaining calories become carbohydrate. Every value can be overridden per client on the Clients tab.
